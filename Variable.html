2. Variables, Data Types & Constants in C

What is a Variable?

  • A variable is like a container in memory where we can store data.
  • Example: a box in which you can keep a number or text.
  • Every variable has:
    1. Name → (identifier)
    2. Type → (int, float, char, etc.)
    3. Value → (the actual data inside it)

Syntax:


         datatype variableName = value;

Example:


         int age = 22;       // integer variable
        float marks = 87.5; // decimal value
        char grade = 'A';   // single character

Data Types in C

C has different data types depending on what kind of value you want to store:

Sr.No Size (Bytes)* Size (Bytes)* Example Value
1 int 2 or 4 10, -50, 200
2 float 4 3.14, 99.5
3 double 8 3.1415926535
3 Char 1 'A', 'z', '1'

What is a Constant?

  • A constant is a value that cannot be changed during program execution.
  • Example: PI = 3.14
  • 2 ways to define constant:

1.Using #define (preprocessor directive)


         #define PI 3.14

Using const keyword


         const int MAX = 100;

Example Program:


        #include <stdio.h>
            #define PI 3.14   // constant using #define

            int main() {
                int age = 22;          // variable
                float marks = 87.5;    // variable
                const char grade = 'A'; // constant using const

                printf("Age: %d\n", age);
                printf("Marks: %.2f\n", marks);
                printf("Grade: %c\n", grade);
                printf("Value of PI: %.2f\n", PI);

                return 0;
            }

Output:


Age: 22
Marks: 87.50
Grade: A
Value of PI: 3.14
