Control Statements in C (if-else, switch-case)

What are Control Statements?

Control statements decide which part of the program will run based on some conditions.

  • Agar condition true hai → ek code chalega
  • Agar false hai → dusra code chalega

1. if Statement

  • Used to check a condition.
  • Agar condition true hai tabhi code chalega.

Syntax:


        if (condition) {
        // code runs if condition is true
        }

Example:


        #include <stdio.h>
        int main() {
        int age = 20;

        if (age >= 18) {
        printf("You can vote!\n");
        }

        return 0;
        }

2. if-else Statement

  • Agar condition true hai → if block chalega
  • Agar false hai → else block chalega

Syntax:


        if (condition) {
        // code if true
        } else {
        // code if false
        }

Example:


        #include <stdio.h>
        int main() {
            int age = 15;

            if (age >= 18) {
            printf("You can vote!\n");
            } else {
            printf("You are too young to vote.\n");
            }

            return 0;
        }

3. if-else-if Ladder

  • Multiple conditions check karne ke liye use hota hai.

Example:


        #include <stdio.h>
        int main() {
            int marks = 75;

            if (marks >= 90) {
                printf("Grade: A\n");
            } else if (marks >= 60) {
                printf("Grade: B\n");
            } else if (marks >= 40) {
                printf("Grade: C\n");
            } else {
                printf("Fail\n");
            }

            return 0;
        }

3. switch-case Statement

  • Jab ek variable ke multiple values check karni ho, switch-case use hota hai.
  • Yeh if-else-if ka easy version hai.

Syntax:


            switch (expression) {
                case value1:
                // code
                break;
                case value2:
                // code
                break;
                default:
                // code
            }

Example:


            #include <stdio.h>
            int main() {
                int day = 3;
                
        switch (day) {
            case 1:
                printf("Monday\n");
                break;
                case 2:
                printf("Tuesday\n");
                break;
                case 3:
                printf("Wednesday\n");
                break;
            case 4:
            printf("Thursday\n");
            break;
            case 5:
            printf("Friday\n");
            break;
            case 6:
            printf("Saturday\n");
            break;
            case 7:
            printf("Sunday\n");
            break;
            default:
            printf("Invalid day\n");
        }
        
        return 0;
    }

Output Examples

  • If age = 20 → You can vote!
  • Marks = 75 → Grade: B
  • Day = 3 → Wednesday

Ab readers easily samajh gaye ki:

  • if → single condition
  • if-else → true/false
  • if-else-if → multiple conditions
  • switch-case → multiple values ke liye
