Loops in C (for, while, do-while)

What is a Loop?

A loop is used to repeat a block of code multiple times until a condition is true/false. Instead of writing the same code again and again, we use loops.

C has 3 main types of loops:

  1. For loop
  2. while loop
  3. do-while loop

1. for Loop

  • Used when we know how many times we want to repeat.

Syntax:


            for(initialization; condition; update) {
            // code to repeat
            }

Example: Print numbers from 1 to 5


        #include <stdio.h>
        int main() {
            for(int i = 1; i <= 5; i++) {
                printf("%d\n", i);
            }
            return 0;
        }

Output:


            1
            2
            3
            4
            5

2. while Loop

  • Used when we don’t know the exact number of times but loop continues until condition is false.

Syntax:


            while(condition) {
            // code to repeat
            }

Example: Print numbers from 1 to 5


        #include <stdio.h>
        int main() {
            int i = 1;
            while(i <= 5) {
                printf("%d\n", i);
                i++;
            }
            return 0;
        }

Output:


            1
            2
            3
            4
            5

3. do-while Loop

  • Similar to while loop, but it always runs the code at least once, even if the condition is false.

Syntax:


            do {
                // code to repeat
            } while(condition);

Example: Print numbers from 1 to 5


        #include <stdio.h>
        int main() {
            int i = 1;
            do {
                printf("%d\n", i);
                i++;
            } while(i <= 5);
            return 0;
        }

Output:


            1
            2
            3
            4
            5

Key Points

  • for loop → Best when number of repetitions is known.
  • while loop → Best when repetitions depend on a condition.
  • do-while loop → Best when you want the loop to run at least once.
