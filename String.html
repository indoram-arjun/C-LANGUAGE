String in C

What is an String?

A string is a collection of characters, stored in an array of characters and ending with a null character (\0).

Example:


         char name[] = "Arjun";
  • Characters = A, r, j, u, n
  • Last automatically = \0

Declaration of String

There are two common ways:


            char str1[] = "Hello";         // automatic \0 added
            char str2[6] = {'H','e','l','l','o','\0'}; // manual

Printing a String

We can store values directly:

Example


            #include <stdio.h>
            int main() {
                char name[] = "C Language";

                printf("%s", name);  // %s is used for string
                return 0;
            }

Output:


           C Language

Taking Input of String


          #include <stdio.h>
        int main() {
            char name[20];
            printf("Enter your name: ");
            scanf("%s", name);   // input string

            printf("Hello %s", name);
            return 0;
        }

If user enters Arjun

Output:


       Hello Arjun

Note: scanf stops at space (so if you enter "Arjun Kumar", it will take only "Arjun").

Common String Functions (from <stdio.h> )

We use string.h library for extra functions:


          #include <stdio.h>
          #include <string.h>
            int main() {
                char str1[20] = "Hello";
                char str2[20] = "World";

                printf("Length of str1: %lu\n", strlen(str1));      // 5
                printf("Copy: %s\n", strcpy(str1, str2));           // World
                printf("Concatenate: %s\n", strcat(str1, str2));    // WorldWorld
                printf("Compare: %d\n", strcmp("abc","abc"));       // 0 (equal)

                return 0;
            }

Key Points

  • Strings are character arrays ending with \0.
  • %s format specifier is used for strings.
  • Use <string.h> functions for length, copy, compare, concatenate, etc.
