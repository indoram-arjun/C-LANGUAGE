Conditional (Ternary) Operator in C

What is Conditional Operator?

  • The conditional operator is a short form of if-else.
  • It is also called the ternary operator because it works with three parts.
  • Symbol: ?:

Syntax:


            condition ? expression1 : expression2;
  • If the condition is true → expression1 runs.
  • If the condition is false → expression2 runs.

Example 1: Check Even or Odd


            #include <stdio.h>
            int main() {
                int num = 10;
                
                (num % 2 == 0) ? printf("Even\n") : printf("Odd\n");
                
                return 0;
            }

Output:


            Even

Example 2: Find Maximum of Two Numbers


            #include <stdio.h>
            int main() {
                int a = 20, b = 15;
                
                int max = (a > b) ? a : b;
                
                printf("Maximum is: %d\n", max);
                
                return 0;
            }

Output:


            Maximum is: 20

Example 3: Pass or Fail


            #include <stdio.h>
            int main() {
                int marks = 35;
                
                (marks >= 40) ? printf("Pass\n") : printf("Fail\n");
                
                return 0;
            }

Output:

     Fail

Key Points

  • TIt is a short and simple alternative to if-else.
  • Use it only for simple conditions.
  • For complex conditions, normal if-else is better.
