1. Introduction to C Language + First Program

Variable: A name that stores some data in memory.

What is C?
  • C is a general-purpose programming language.
  • It was created by Dennis Ritchie in 1972.
  • It is fast and used for system programming (like operating systems).

First Program – Hello, World!


      #include<stdio.h>

        int main() {
            printf("Hello, World!\n");
            return 0;
        }

C Language Boilerplate Code

Variable: A name that stores some data in memory.

Simple Explanation

This is a library (toolbox) in which input/output functions (like printf, scanf) are already ready. Without this, you cannot print or take input.

Every C program starts from here. main() is the starting point of the program.

printf means “print formatted output”. It shows text on the screen. \n means new line.

Tells the computer: “Program finished successfully.”

Example Code


      #include<stdio.h>// Header file

      int main() {
          printf("Hello, C Programming!\n");   // Print text on screen
          return 0;   // End program
      }
