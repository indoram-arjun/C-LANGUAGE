Array in C

What is an Array?

An array is a collection of similar data stored at continuous memory locations. ðŸ‘‰ It is like a box with many compartments where each compartment stores one value.

Example: Instead of writing 5 variables:


         int a1, a2, a3, a4, a5;

We can use an array:


         int arr[5];

Declaration of Array


         data_type array_name[size];

Example:


        int numbers[5];   // array of 5 integers

Initialization of Array

We can store values directly:


          int numbers[5] = {10, 20, 30, 40, 50};

Accessing Array Elements

  • Each element has an index number (position).
  • Index starts from 0.

Example:


          #include <stdio.h>
            int main() {
                int numbers[5] = {10, 20, 30, 40, 50};

                printf("First element: %d\n", numbers[0]);  // 10
                printf("Second element: %d\n", numbers[1]); // 20
                printf("Last element: %d\n", numbers[4]);   // 50

                return 0;
            }

Output:


        First element: 10
        Second element: 20
        Last element: 50

Using Loop with Array

We can print all elements using a loop:


          #include <stdio.h>
            int main() {
                int numbers[5] = {10, 20, 30, 40, 50};

                for(int i = 0; i < 5; i++) {
                    printf("%d\n", numbers[i]);
                }

                return 0;
            }

Output:


        10
        20
        30
        40
        50

Key Points

  • Arrays store multiple values of same type.
  • Index starts from 0.
  • Useful for storing large data like marks of students, salaries, etc.
